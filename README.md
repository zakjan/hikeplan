# HikePlan

Hiking route planner

[![CircleCI](https://img.shields.io/circleci/project/zakjan/hikeplan.svg)](https://circleci.com/gh/zakjan/hikeplan)

## Dependencies

```
npm install -g webpack
npm install
```

## Development

```
webpack -w
npm start
```

=> [http://localhost:3000](http://localhost:3000)
